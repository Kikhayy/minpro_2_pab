# Aplikasi Data Resep Masakan #

# Deskripsi Aplikasi
Aplikasi Data Resep Masakan adalah aplikasi sederhana berbasis Flutter yang digunakan untuk mengelola data resep masakan. Pengguna dapat menambahkan, melihat, mengedit, dan menghapus data resep masakan. Setiap resep memiliki tiga data utama yaitu nama resep, bahan, dan cara memasak.

# Fitur Aplikasi
1. Create (Tambah Data Resep)
Pengguna dapat menambahkan resep baru dengan mengisi nama resep, bahan, dan cara memasak melalui tombol + yg tersedia di pojok kanan halaman utama.
